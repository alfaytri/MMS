Contracts Module — Complete Breakdown
Overview
The Contracts module has 4 pages and 2 distinct workflows:

Contract Quotations — creating and approving new contract proposals
Live Contracts — managing active contracts with visit scheduling and payment tracking

Contract Types (TypeScript)

type ContractLiveStatus = "active" | "expiring_soon" | "overdue_payment" | "cancelled" | "completed";

interface LiveContract {
  id, contractId, customerName, phone, address, siteName,
  divisions: string[],
  servicesSummary, agentName,
  startDate, endDate,
  status: ContractLiveStatus,
  monthlyValue, totalValue,
  totalVisits, completedVisits,
  upcomingVisits: { date, serviceName, teamName? }[],
  totalPayments, paidAmount,
  payments: { dueDate, amount, status: "paid"|"pending"|"overdue" }[],
  paymentSchedule, hasSignedDoc, areaCount,
  cancelledDate?, cancelReason?
}
